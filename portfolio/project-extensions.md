# Project Extensions

Pick one extension so your portfolio project is not an identical clone.

## Beginner Extensions

- Add `sigmoid`.
- Add binary cross entropy.
- Add dropout.
- Add model save/load.
- Add more tests for broadcasting and shape edge cases.

## Intermediate Extensions

- Add a tiny convolution layer.
- Add layer normalization.
- Add a dataloader.
- Add gradient checking with finite differences.
- Compare training curves against PyTorch on the same model.

## Advanced Extensions

- Add GPU support through CuPy.
- Add a small computation graph visualizer.
- Add a mini transformer block.
- Add mixed precision experiments.
- Add a benchmarking report against PyTorch for tiny models.
