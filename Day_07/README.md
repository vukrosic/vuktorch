# Day 07 - Loss Functions

Day 07 turns model outputs into a training signal.

## Goal

Implement MSE first, then build intuition for classification loss with
cross-entropy.

## Build

- mean squared error
- cross-entropy
- loss aggregation across a batch

## Why It Matters

Without a loss, the model has no direction. This day makes the framework
optimizable.

## Exercise

Compare MSE and cross-entropy on the same toy predictions.
