# Day 07 - Loss Functions

Day 07 turns model outputs into training signals.

## Goal

Implement MSE first, then build intuition for classification loss with
cross-entropy.

## Build

- mean squared error
- cross-entropy
- batch aggregation

## Why It Matters

Without a loss, gradients have no direction. This lesson gives learning a
target.

## Exercise

Compare MSE and cross-entropy on the same toy predictions.
