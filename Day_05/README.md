# Day 05 - Modules and Neurons

Day 05 introduces the object model for learned components.

## Goal

Introduce a `Module` base class and a single vector-input neuron abstraction.

## Build

- `Module`
- `Neuron`

## Why It Matters

Once tensors work, the next job is organizing parameters and computation into
reusable pieces.

## Exercise

Build a neuron with a few scalar inputs and inspect its parameters.
